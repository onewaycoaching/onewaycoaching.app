# OneWay legal pages — GitHub Pages deploy

**Work item:** WQ-018

## Deploy (Nicholas — requires approval)

1. Create repo e.g. `oneway-coaching.github.io` or add `/docs` folder to existing org repo
2. Copy `privacy.html`, `terms.html`, `support.html` to publish root
3. Enable GitHub Pages (Settings → Pages → main branch /docs)
4. Replace `[DATE]` and confirm `support@onewaycoaching.com` is live
5. App Store Connect URLs:
   - Privacy: `https://[user].github.io/privacy.html`
   - Support: `https://[user].github.io/support.html`

Merlin must ask before `git push` or any publish.
